Sign Up

Um eine Waschmaschine hinzufügen zu können müssen sie sich einen Github Account anlegen, da das System Github als Datenspeicher benutzt.

Erstellen eines Repository

Der Datenspeicher wird in Form einer Textdatei gespeichert werden, somit muss zuerst ein Ordner (Repository) angelegt werden. In diesem Ordner werden später die verschiedenen Waschmaschinen als einzelne Textdateien angelegt werden.

1 Geben sie einen Namen für das Repository ein
2 Wähle "Initialize this repository with a README" aus
3 Sie können das Repository optional noch eine Beschreibung hinzufügen
Hinzufügen einer Waschmaschine

Diese App benötigt für das hinzufügen einer Waschmaschine einen Raw-Link dieser wird nach erzeugen der leeren Textdatei, durch klicken auf den Button "Raw" erzeugt. Kopiere dazu den Link in ihrer Suchleiste.

1 Wähle "Create new file" in deinem Repository aus
2 Gib der Waschmaschine einen Namen deiner Wahl
3 Klicke "Commit new file"
4 Wähle "Raw" um nur die Datei anzuzeigen
5 Um eine neue Waschmaschine hinzuzufügen, klicke das Symbol
6 Wähle einen Namen deiner Wahl und kopiere den Raw-Link in das dafür vorgesehene Feld und klicke den Button "Add" um die Waschmaschine hinzufügen
