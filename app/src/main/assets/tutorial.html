Erstellen eines Repository
1. Gebe dem Repository einen Namen
2. Beschreibe es wenn du willst
3. Wähle "Initialize this repository with a README" nicht aus
Hinzufügen einer Waschmaschine
1. Wähle "Create new file" in deinem Repo aus
2. Benenne die Waschmaschine wie du willst
3. Drücke "Commit new file"
4. Wähle "Raw" bei deiner txt Datei aus
5. Wähle das Symbol aus
6. Wähle einen Namen und kopiere deinen Raw Link in das freie Feld und drücke den Knopf "Waschmaschine hinzufügen"
